### 쉴드 이벤트 UI 연동
- **상태**: 완료
- **관련 파일**:
  - `Assets/02.Scripts/Combat/BattleSceneSetup.cs` — OnShieldChanged 이벤트 구독 추가
- **내용**:
  - HealthComponent.OnShieldChanged 이벤트가 어디에서도 구독되지 않아 쉴드 변화가 UI에 반영되지 않던 버그 수정
  - 모든 캐릭터(플레이어/적)의 OnHPChanged + OnShieldChanged를 동일 핸들러에 구독
  - OnCharacterHPChanged → OnCharacterHealthChanged로 명명 변경
- **결과**: 쉴드 추가/소모 시 실시간 UI 갱신, 쉴드 완전 흡수 시에도 HP 변동 없이 쉴드 바만 감소 표시

### 쉴드 바 시각화 (HP 바 위 갈색 오버레이)
- **상태**: 완료
- **관련 파일**:
  - `Assets/02.Scripts/Editor/BattleUISceneBuilder.cs` — ShieldBrown 색상 상수 추가
  - `Assets/02.Scripts/Editor/BattleUISceneBuilder.UI.cs` — 플레이어/적 패널 HP 바에 ShieldFill 이미지 추가
  - `Assets/02.Scripts/UI/Battle/PlayerSidebarPanel.cs` — _shieldFillImage 필드, 자동 와이어, UpdateHP 쉴드 바 로직
  - `Assets/02.Scripts/UI/Battle/EnemyDetailPanel.cs` — 동일
- **내용**:
  - HP 바 내부에 ShieldFill 이미지를 Fill 위에 겹쳐 배치
  - 쉴드 바 anchorMin.x = HP 비율 (HP 바 끝점), anchorMax.x = min(1, HP+쉴드/MaxHP)
  - 쉴드 0이면 숨김, 쉴드 있으면 HP 바 끝에서부터 갈색 바로 표시
  - 최대 체력바 길이(100%) 초과하지 않도록 Mathf.Min(1f, ...) 제한
- **결과**: HP 20/20 + 쉴드 4 → 초록 100% 위에 갈색 바 HP 비율(100%)~쉴드 비율(120%→100%) 표시

### 스킬 최종 위력 표시 (ATK + Power)
- **상태**: 완료
- **관련 파일**:
  - `Assets/02.Scripts/Combat/AI/EnemyIntentType.cs` — FromSkill에서 ATK+Power를 최종 위력으로 저장, GetDisplayText에서 Value 사용
  - `Assets/02.Scripts/UI/Battle/ActionBarUI.cs` — BuildActionDescription 추가 (최종 위력/쉴드/회복/상태이상)
  - `Assets/02.Scripts/UI/Battle/CharacterPopupUI.cs` — BuildSkillSummary에 ATK+Power 반영
- **내용**:
  - 데미지 공식 `max(1, ATK + Power - DEF)`에서 실제 위력은 `ATK + Power`
  - 적 의도 패널: "몸통박치기 2 → 전사" (기존: 위력 1만 표시 → 수정: ATK+Power 합산)
  - 액션 바 스킬 설명: "기본 설명 | 최종 위력 2 | 독 1 (3턴)"
  - 캐릭터 팝업 스킬 목록: "파이어볼 — 위력 4 | 화상 1 (2턴)"
- **결과**: 모든 UI에서 플레이어가 실제 데미지를 예측 가능
