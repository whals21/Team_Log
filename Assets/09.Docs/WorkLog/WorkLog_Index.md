## 작업일지 작성 규칙

1. **파일명**: `YYYY-MM-DD.md` (하루에 여러 작업이면 같은 파일에 추가)
2. **기간 작업**: `YYYY-MM-DD_YYYY-MM-DD.md` (여러 날에 걸친 작업)
3. **형식**: 각 작업 항목은 아래 템플릿 사용

```
### [작업명]
- **상태**: 완료 / 진행 중 / 보류
- **관련 파일**: [파일 경로 목록]
- **내용**: [무엇을 했는지]
- **결과**: [어떤 결과가 나왔는지]
- **다음 단계**: [이어서 해야 할 작업]
```
